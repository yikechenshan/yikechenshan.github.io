JDK、JRE

使用jdk的开发工具完成的java程序，交给jre运行。

JDK

java development kit java开发工具包

包含java开发工具，也包含了jre

JRE

java runtime environment java运行环境

包含java虚拟机(JVM)和java程序所需的核心类库

想要运行一个开发好的java程序，计算机中只需要安装jre。

总结

JDK = JRE + 开发工具集（例如javac编译工具等）

JRE = JVM + Java SE 标准类库

简易方法
