装好arch之后总得施工装修一下，虽然不施工又不是不能用，话说回来，没有中文输入法好像真的不大行，其实在装系统的时候可以顺带把中文输入法装了的。。。

安装zsh

关闭终端，安装字体

MesloLGS NF Regular.ttf

MesloLGS NF Bold.ttf

MesloLGS NF Italic.ttf

MesloLGS NF Bold Italic.ttf

点击下载，然后打开安装

打开终端->设置->管理配色方案->新建->外观->字体->选择MesloLGS NF
将新的配色方案设置为默认

重启终端，出现powerlevel10k的设置界面，按照提示进行设置

中文界面

System Setting->Regional Settings->Language->add language

添加简体中文后将简体中文拖到第一位

重启
