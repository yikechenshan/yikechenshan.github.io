一、 解压软件的选择

不推荐:快压，好压，2345 压缩王等等一系列花里胡哨的解(流)压(氓)软件。

推荐:7-zip 等功能全，体积小，无捆绑、无广告的解压软件。

(据称很多花里胡哨的解压软件用的就是 7-zip 内核，然后套壳花里胡哨的东西)

二、 解压软件的安装

*安装过程在一个全新刚安装的 64 位 Window10 操作系统上进行演示(下同)。

2.1 下载

由于是国外网站，可能出现无法访问等情况，可以通过 QQ 群文件下载，或通过微云下载

微云链接:https://share.weiyun.com/Hkz2VULj 密码:5dqgm5

官网下载方式如下：

2.1.1 在浏览器中输入 7-zip 进行搜索。

2.1.2 点击第一个搜索结果，进入官网之后(www.7-zip.org)，点击红框中的链 接下载 64 位正式版。

2.1.3 弹出保持文件的窗口，点击【保存】。

2.2 安装

2.2.1 点击左下角的【打开文件】。

2.2.2 选择【是】

2.2.3 点击【Install】(可以自己选择安装路径，推荐不用改)

2.2.4 安装完毕点击【Close】

2.3 配置

2.3.1 打开开始菜单，点击【7-zip File Manager】

2.3.2 鼠标指向【工具】，然后鼠标左击【选项】

2.3.3 点击第一个【+】号

2.3.4 蓝框中的内容显示如图后，点击【应用】

2.3.5 点击【确定】，然后再关闭软件，配置完成

2.4 使用

2.4.1 右击要操作的文件，选择<7-zip>，再选择要进行的操作即可。
